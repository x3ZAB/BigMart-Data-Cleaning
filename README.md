# Big Mart Sales Data Cleaning

This project focuses on cleaning and preparing the Big Mart Sales dataset using Python and Pandas.

## Data Cleaning Steps
- Handled missing values
- Standardized categorical values
- Fixed zero values in Item_Visibility
- Removed duplicates
- Created Outlet_Age feature

## Files
- bigmart_data_cleaning.ipynb
- cleaned_bigmart.csv

## Tools Used
- Python
- Pandas
- Jupyter Notebook
